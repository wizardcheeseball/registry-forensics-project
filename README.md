# Windows Registry Analysis Project

## Overview
This project demonstrates analysis of Windows Registry artifacts to identify system activity, persistence mechanisms, and potential security risks.

## Objectives
- Identify key registry locations related to system activity
- Analyze artifacts for evidence of persistence or unusual behavior
- Document findings and security implications

## Tools Used
- Windows Registry Editor
- Autopsy (if used)
- CyberChef (if applicable)

## Key Findings
- Identified registry keys related to startup persistence
- Observed system configuration and user activity artifacts
- Documented potential security risks and mitigation considerations

## Skills Demonstrated
- Digital forensics fundamentals
- System artifact analysis
- Documentation and reporting
